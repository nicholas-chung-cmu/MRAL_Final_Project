MRAL Final Project

A simple kinematic planner utilizing Signed Distance Field (SDF) Grids.

Files:
- robot.py: defines Robot object; contains updated code for path-planning algorithm with a Robot object.
- sdf.py: defines SDF object; contains original code for path-planning algorithm.
- plotting_functions.py: contains functions related to plotting figures to visualize results of sdf grid and path planning generation.
- sdf_test.py: runs test cases.
